Welcome to the den

This is a minimalist corner of the web inspired by the tranquil Bear Blog aesthetic. It keeps things light, fast, and focused on thoughtful writing. The whole site is a handful of plain files, which means you can host it anywhere—including GitHub Pages.

Use this as a starting point for your own stories, field notes, and experiments. Publish a new post by duplicating one of the examples inside the posts/ folder and updating the metadata. The navigation updates instantly because the site is static—nothing to recompile.

Ready to explore? Head to the posts index for a tour of existing entries, or learn more about the author on the about page.
